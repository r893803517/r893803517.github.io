# DOM文档对象模型

是针对HTML和XML文档的一个API，将HTML或XML文档描绘成一个层次化的节点树


在 HTML DOM 中 , 每一个元素都是 节点:

> 文档是一个文档节点。

> 所有的HTML元素都是元素节点。
 
> 所有 HTML 属性都是属性节点。
 
> 文本插入到 HTML 元素是文本节点。

> 注释是注释节点。
